# Paradise Nursery

## Project Overview
**Paradise Nursery** is a front-end e-commerce application built with React. This application allows users to browse a variety of beautiful houseplants, organize them by categories, and manage their purchases through a dynamic shopping cart.

This project was developed as part of a final project assessment to demonstrate proficiency in React, Redux for state management, and modern web design principles.

## Features
* **Landing Page:** A welcoming entry page featuring the company name, background imagery, a brief "About Us" section, and a call-to-action button to start shopping.
* **Product Listing Page:** Displays at least six unique houseplants categorized into distinct groups (e.g., Air Purifying, Succulents, Tropical). Users can add items to their cart directly from this page.
* **Dynamic Shopping Cart:** 
  * Powered by Redux Toolkit for seamless state management.
  * Displays a dynamic cart icon in the navigation bar that updates the total item count in real-time.
  * Allows users to increase, decrease, or remove items entirely from their cart.
  * Calculates subtotal and total costs automatically.
* **Responsive Navigation:** A persistent header that allows easy switching between the Home, Plants, and Cart pages.

## Technologies Used
* **React.js** (Functional Components & Hooks)
* **Redux Toolkit** (State management for the shopping cart)
* **React Router DOM** (Client-side routing)
* **CSS3** (Custom styling and flexbox layouts)
